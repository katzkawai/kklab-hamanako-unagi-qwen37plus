# 浜名湖うなぎ サイト

静岡・浜名湖のうなぎ店舗・ブランド紹介サイトです。
伝統的な和の雰囲気を取り入れた、シンプルでレスポンシブなデザインを採用しています。

## 技術スタック

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
